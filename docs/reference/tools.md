# Tool Registry

Built-in tools available in CIDX.

## Security Phase

- **trivy**: Vulnerability scanner (aquasec/trivy)
- **gitleaks**: Secret detection (zricethezav/gitleaks)

## Code Phase

- **megalinter**: MegaLinter (oxsecurity/megalinter)
- **prettier**: Code formatter (tmknom/prettier)
- **commitlint**: Commit message linter (commitlint/commitlint)
- **golangci-lint**: Go linter (golangci/golangci-lint)
- **ansible-lint**: Ansible linter (ansible/ansible-lint)
- **shellcheck**: Shell script static analysis (koalaman/shellcheck)
- **gofmt**: Go formatter

## Test Phase

- **go-test**: Go unit testing
- **godog**: BDD framework for Go
- **molecule**: Ansible testing

## Build Phase

- **go-build**: Go compiler
- **docker-buildx**: Docker Buildx (multi-platform)

## Release Phase

- **gh-release**: GitHub Releases (cli/cli)
- **goreleaser**: GoReleaser (goreleaser/goreleaser)
- **kaniko**: Container image builder (GoogleContainerTools/kaniko)
